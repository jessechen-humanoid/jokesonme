# common-fund Design

## Overview

在財務分析頁面新增「看我笑話共同基金」區塊，並調整成員年度報表的計算邏輯，將共同淨利的 20% 提撥為團隊儲備金。

## Key Definitions

- **共同交易**：`excludedMembers === ''`（沒有排除任何成員）的交易
- **非共同交易**：`excludedMembers !== ''`（有排除成員）的交易
- **共同基金提撥率**：20%

## Calculation Logic

```
共同收入 = Σ amount（amount > 0 且 excludedMembers === ''）
共同支出 = Σ amount（amount < 0 且 excludedMembers === ''）
共同淨利 = 共同收入 + 共同支出
共同基金 = 共同淨利 × 0.2
可分配共同淨利 = 共同淨利 × 0.8
```

### Per-Member Calculation

```
成員年度分配淨利 =
  （可分配共同淨利 ÷ 8）          ← 共同部分，先扣基金再均分
  ＋ 非共同部分的個人份額            ← 照原本 excludedMembers 邏輯
```

非共同部分的個人份額 = 該成員在所有 `excludedMembers !== ''` 交易中被分配到的金額總和。

## Data Flow

```
Google Sheets（收支紀錄）
    ↓
API: getTransactions()
    ↓
Frontend: js/analytics.js
    ├─ 分類交易：共同 vs 非共同（by excludedMembers）
    ├─ 計算共同基金提撥
    ├─ 計算每人年度分配淨利
    └─ 渲染：
        • 共同基金區塊（新）
        • 成員年度報表（改：年度淨利 → 年度分配淨利）
```

## UI Layout

共同基金區塊放在「年度總覽」與「各場次損益」之間：

```
┌─────────────────────────────────┐
│         年度總覽                 │
│  總收入 | 總支出 | 淨利          │
├─────────────────────────────────┤
│      看我笑話共同基金            │  ← 新區塊
│  共同收入 | 共同支出 | 共同淨利   │
│  提撥 20%：$XX,XXX              │
├─────────────────────────────────┤
│         各場次損益               │
│         ...                     │
├─────────────────────────────────┤
│         成員年度報表             │
│  欄位：年度分配淨利（原年度淨利） │
└─────────────────────────────────┘
```

## Google Sheet

在 Google Sheet 中需要能看到共同基金數字。方式：
- 在現有的報表輸出中加入共同基金相關欄位
- 或在 GAS 的 getAnalytics 回傳中包含共同基金計算結果

## Edge Cases

- 共同淨利為負數時：共同基金 = 負數 × 20% = 負數（代表基金需要支出）→ 仍照公式計算
- 沒有任何共同交易時：共同基金 = 0
- 全部都是共同交易時：等於整體淨利的 20%
